timepicker-ui

timepicker-ui is an easy library with timepicker. Fully wrote with TypeScript. This library is based on Material Design from Google.

These are examples of use timepicker-ui library. Fully wrote documentation you can find here or on the GitHub page.

This site is using Bootstrap 4, font Roboto and material-icons from Google.

HTML
                
          <div class="basic" >
            <input
              type="text"
              class="timepicker-ui-input"
              value="12:00 PM"
            />
          </div>
JavaScript
                
          import { TimepickerUI } from "timepicker-ui";

          const basic = document.querySelector('.basic');
          const basicPicker = new TimepickerUI(basic, {});
          basicPicker.create();
